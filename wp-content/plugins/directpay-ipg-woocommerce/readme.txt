== Installation ==

1. Download the DirectPay Plugin for WooCommerce from Wordpress Plugin Directory
2. Upload the 'woocommerce-directpay' folder to your WordPress 'plugins' directory under '/wp-content/plugins'
3. In your WordPress admin, go to 'Plugins' and Activate the 'WooCommerce DirectPay Payment Gateway' plugin
4. Go to your WooCommerce > Settings page, click on 'Checkout' tab & click the ' DirectPay' link
5. Tick the checkbox to Enable DirectPay
6. Enter the payment title and description or keep the default
7. Enter your Merchant ID (You can find it on the Settings > DirectPay Plugins section of your DirectPay account)
8. Enter your API Key (You can find it on the Settings > DirectPay Plugins section of your DirectPay account)
9. Do not tick the checkbox to Enable Sandbox Mode (Tick it only if you're testing your integration with a DirectPay Sandbox Account)
10. Click Save Changes
